# Codexに貼るプロンプト集

このページは、学生が授業中や宿題中にそのまま使えるプロンプトの原稿です。

`AGENTS.md` に、自分の前提や説明してほしい細かさは書いてあります。そのため、毎回のプロンプトでは「何を作りたいか」よりも、「どんな課題に対して、どんな解決策を出したいか」を具体的に書きます。

## 動かし方を聞く

```text
今作ったものを自分のPCで開いて確認したいです。
私は初心者なので、次に何をすればよいかを1手順ずつ書いてください。
コマンドが必要なら、どこに貼るのかも説明してください。
最後に、ブラウザで開くURLも教えてください。
```

## エラーが出たとき

```text
次のエラーが出ました。
私は意味が分からないので、原因の候補と、次に試すことを順番に教えてください。
必要なら修正もしてください。

エラー:
ここに表示された文字をそのまま貼る
```

## 画面を見て直す


## 文字が多すぎるとき

```text
今のページは文字が多く、最初にどこを見ればよいか分かりにくいです。
内容は大きく削りすぎず、見出し、余白、並び順を調整して、読みやすくしてください。
最初に伝わるべき内容が一番目立つようにしてください。
```

```text
5. 何ができるか
```
