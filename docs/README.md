# Codexで「ほとんどなんでも作れる」感覚を持つ授業資料ドラフト

この `docs/` フォルダは、Web教材にする前の編集用Markdown置き場です。

- `curriculum.md`: 授業全体の目的、ページ構成案
- `workflow.md`: どの事例でも共通する制作フロー
- `examples.md`: 学生がたくさん作ってみるための事例集
- `prompt-templates.md`: 学生がCodexにそのまま貼れるプロンプト集
- `student-agents-template.md`: 学生が自分用に調整して使う `AGENTS.md` の原稿

対象は、商学部3年生で、コンピュータやプログラミングにあまり馴染みがない学生です。
そのため、教材では専門用語をできるだけ減らし、「何を押すか」「何を聞くか」「困ったら何を貼るか」を具体的に書く方針にしています。

授業内で自分を表すWebサイトも作りますが、それは最終目標ではなく、Codexで何ができるのかを体験するための練習例の一つです。
授業は講義形式ではなく、学生がページを見ながら各自で進め、分からないところを質問する形式を想定しています。
`AGENTS.md` は作業フォルダごとに置く前提です。画像を使う本番制作では、Computer Use + 編集で自分で作る、または編集した画像に差し替える方針です。
今日作ったものは、最後にスクリーンショットを撮ってDiscordに投稿する想定です。
