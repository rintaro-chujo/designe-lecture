## 今日の最後にやること

1. 作ったものをブラウザで開く
2. 画面が見える状態でスクリーンショットを撮る
3. Discordの指定チャンネルに投稿する
4. できれば、「何を作ったか」、「工夫したところはどこか」、「一言コメント」を添える

投稿例:

```text
インスタのストーリー風アプリを作りました。背景色と短いテキストを変えられるようにしました。エラーがたくさん出て難しかったです
```

## まだ動いていない人

今日の時点で完全に動いていなくても、途中経過を投稿して構いません。

その場合は、どこまでできたかを書いて、スクリーンショットとともに提出してください。

```text
相性診断風アプリを作っています。まだ結果画面がうまく出ませんが、質問画面まではできました。
```
